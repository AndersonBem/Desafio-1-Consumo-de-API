# 🧠 Sobre o projeto

O objetivo da aplicação é demonstrar o **consumo de uma API pública utilizando JavaScript**, realizando requisições HTTP e exibindo os dados dinamicamente na interface.

A aplicação:

- realiza requisições utilizando **Fetch API**
- consome dados de uma **API pública**
- exibe as informações de forma organizada na página
- possui layout **responsivo**, priorizando visualização em **dispositivos móveis**

# 🌐 API utilizada

O projeto utiliza a API:

**NASA Astronomy Picture of the Day (APOD)**

Documentação oficial:

https://api.nasa.gov/

A API retorna dados como:

- título da imagem
- descrição
- data
- URL da imagem
- outras informações relacionadas à imagem astronômica do dia

# 🚀 Como executar o projeto localmente

1. Clone o repositório

```bash
git clone https://github.com/AndersonBem/Desafio-1-Consumo-de-API.git
```

2. Acesse a pasta do projeto

```bash
cd Desafio-1-Consumo-de-API
```

3.Abra o arquivo index.html no navegador.

Não é necessário instalar dependências.
